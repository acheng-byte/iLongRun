# iLongRun v0.1.0 发版说明

## 这是什么

iLongRun 是一个面向 GitHub Copilot CLI 的长跑蜂群编排插件，目标是让复杂任务不再只停留在单层 plan，而是具备：
- 顶层 plan
- 多层 task list
- 可恢复状态机
- GPT-5.4 coding 终审
- `/fleet` 波次级并行执行能力

## 首发能力

- 独立插件与独立仓库
- 一键安装
- 五种模式：Direct Lane / Wave Swarm / Super Swarm / Fleet Governor / Sentinel Watch
- coding 任务强制 GPT-5.4 终审
- `/fleet` 探测、dispatch、自动降级

## 安装方式

```bash
curl -fsSL https://raw.githubusercontent.com/izscc/iLongRun/main/install.sh | bash
```

## 核心命令

```bash
ilongrun
ilongrun-prompt
ilongrun-resume
ilongrun-status
ilongrun-doctor
copilot-ilongrun
```

## 已知限制

- `/fleet` 是否可用依赖本机 Copilot CLI 的真实能力
- 部分环境下插件注册成功但 hooks 可能因本机配置差异未启用，此时不影响核心功能
